# 名词解释

## 宿主

宿主指的是使用编辑器的环境，一般是集成编辑器的项目。

## 生产者
`模块联邦`中的概念，在这里代表由开发人员编写的业务组件。

## 消费者
`模块联邦`中的概念，在这里代表集成项目引入的编辑器，它负责根据用户在编辑区布局的组件从生产者中获取对应的组件进行渲染。

## 扩展
参考了`LowCodeEngine`的设计，支持通过`模块联邦`的形式引入功能扩展。

## 布局

如下图所示，整个编辑器可以分为三个部分：组件区，编辑区以及配置区。

![布局](./assets/layout.png)

### 组件区

组件区用于展示所有组件，支持分组。可以直接拖拽到编辑区进行放置。

### 编辑区

编辑区用于用于调整组件在页面中的位置，支持拖拽排序。点击后可以选中组件在配置区进行进一步设置。

### 配置区

配置区用于设置组件的各项属性以及样式。
